## 器材准备

- Pico_DM_1P5623 显示拓展板
- 树莓派 Pico 核心板
- USB 数据线
- USB 转串口模块

## 开发环境搭建

:::note
本项目默认使用 **Ubuntu/Debian** 作为开发系统。
:::

### 部署 pico-sdk

<Steps>

1. 安装前置依赖

      ```bash
      sudo apt install cmake gcc-arm-none-eabi libnewlib-arm-none-eabi libstdc++-arm-none-eabi-newlib ninja-build
      ```

2. 下载 pico-sdk 源码

      根据网络情况选择下载源。Gitee 是 GitHub 的镜像

      ```bash title="gitee"
      git clone https://gitee.com/embeddedboys/pico-sdk $HOME/.pico-sdk
      ```

      ```bash title="github"
      git clone https://github.com/raspberrypi/pico-sdk/ $HOME/.pico-sdk
      ```

3. 拉取 pico-sdk 子仓库代码

      ```bash
      cd ~/.pico-sdk
      git submodule update --init
      ```

4. 配置环境变量

      **以 bash 为例**

      ```bash
      echo "export PICO_SDK_PATH=$HOME/.pico-sdk" >> ~/.bashrc
      source ~/.bashrc
      ```

</Steps>
